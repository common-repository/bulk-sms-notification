Bulk SMS Notification plugin sends SMS in bulk. Bulk SMS Notification sends bulk SMS based on selected user role. 

== Description ==

You can send SMS to the customers worldwide. Send SMS notification in bulk to market your buisness.

The plugun is designed to do marketing via bulk SMS and it is based on the user roles. You just need to select the user role you wish to send the SMS notification to. 

<h4> Free Plugin Features </h4>
* Send Bulk SMS Notification to all the users.
* Send Bulk SMS Notification to administrator role.
* Send Bulk SMS Notification to subscriber role.
* Send Bulk SMS Notification to selected role.

miniOrange uses login.xecurify.com if the user chooses to register and use miniOrange SMS gateway.

== Installation ==


= From your WordPress dashboard =

1. Navigate to `Plugins > Add New` from your WP Admin dashboard.

2. Search for `Bulk SMS Notification`.

3. Install `Bulk SMS Notification` and Activate the plugin.



= From WordPress.org =

1. Search for `Bulk SMS Notification` and download it.

2. Unzip and upload the `Bulk SMS Notification` directory to your `/wp-content/plugins/` directory.

3. Activate Bulk SMS from the Plugins tab of your admin dashboard.


== Frequently Asked Questions ==

= Once Activated =

1. Select miniOrange Bulk SMS from the left menu and follow the instructions.

2. Register/Login with miniOrange.

3. Configure the phone user meta key.

4. Start sending the SMS notifications.
